SDI MES그룹(부품MES)
SDI 차세대 MES Global 운영 체계 수립 및 모니터링 체계 강화 통한 장애 예방

1. 추진 배경
SDI 차세대 MES 전사(5개국, 7개 사업장) 횡전개 완료 후 시스템 조기 안정화를 위해
1) Global 표준 운영 체계 수립
2) SDI 전사 수천대 규모의 서버 실시간 모니터링
3) 이슈 조기 감지, 신속한 상황 전파 및 긴급 조치 체계 수립
하여 시스템-프로세스-전담조직 기반의 장애 예방 체계 구축 필요

2. 추진 결과
1) Global 운영체계 수립
Global 원소스 체계 운영 <- 지원요청/장애전파 -> Global 통합상황실(24*365) <- 이상감시/긴급대응 -> 생산법인
Global 원소스 체계 운영 <- 법인특화 BP사례 요청, 현장점검 피드백 통보, 배포 -> 생산법인

생산법인(9개운영,4개예정): 
- SDSA(SPE, GAMMA, 조지아)
- SDSE(헝가리, 체코)
- SDSC(천진, 시안, 무석)
- SDSAP(말련1, 말련2)
- SDSK(울산, 천안, 청주, 구미)

Global 원소스 체계 운영:
VOC분석/설계(프로세스 표준화) -> 분석/설계 -> 개발/테스트 -> 검수 -> 배포 -> 기술지원/장애대응(표준 MES 배포)

2) 실시간 모니터링 항목 표준화
- 전체 25개: DB 9, H/W 3, Middleware(H101) 5, MOS 6, TC 3

3) SDI 자체 구축 AMS 시스템에 표준화된 모니터링 항목을 등록하여 상시 모니터링 가능

4) 모니터링 항목별 긴급조치 프로세스 매뉴얼화
구분, 모니터링항목(한글/영문), 현상, 영향, 조치(점검, 조치방법)을 정리한 문서

5) Global 운영 배포 모니터링 자동화를 통해 배포 중 이슈 실시간 감지 및 조치 시간 단축
전체 배포 진행률, 상세 배포 진행률 표시의 자동화/모니터링 화면

3. 추진 효과
- 운영 프로세스 및 시스템 운영 표준화로 차세대 MES 조기 안정화(횡전개 이후 장애 발생건 없음)
- Global 실시간 모니터링 및 이슈 조기 감지 및 조치 시간 단축으로 장애 예방 효과 극대화

4. 향후 계획
- 정기 시스템 운영자 교육 (사이트별 2회/년)
- Global 시스템 실시간 모니터링 체계 고도화 ( ~ 2025.12)
